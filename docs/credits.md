## What Ahsan built for ImtehanHub

Every layer:

- **Product design** — feature scope, content hierarchy (Class → Subject → Book → Chapter → Question), bilingual model (`name` + `urduName` everywhere), monetisation strategy (Free / Pro / Unlimited / Add-on), growth loops (referrals + 3-invites-auto-Pro), institute layer with discount tiers
- **Frontend** — React 19 + Vite 8 + Tailwind v4 + Radix UI + Zustand + TanStack Query + RHF + Zod across ~83 docs pages and a similar volume of app pages
- **Native mobile** — Capacitor 8 Android app with offline mode, queue-and-replay submissions, native sign-in, branded status bar, OneSignal push notifications
- **Backend** — Firebase free tier only (Firestore 50K reads/day, Auth, Hosting). No Cloud Functions. No Firebase Storage (user files live in user-owned Google Drive instead)
- **Data pipeline** — Node.js scrapers for free Pakistani exam-prep sites (ilmkidunya.com, gotest.pk), transforms into typed JSON, schema + content validation, Firestore batch-seeding. Tesseract OCR fallback for scanned-image PDFs
- **SEO / AEO infrastructure** — `postbuild-seo.mjs` generates 660+ static HTMLs per build with per-route JSON-LD (Organization, WebSite, Article, FAQPage, HowTo, BreadcrumbList, Course, LearningResource), plus `llms.txt`, `ai.txt`, AI-bot allowlist in `robots.txt`, sitemap.xml, feed.xml
- **Community module** — verified contributor system: CNIC verification via SHA256-hash + Drive-only PII, submissions (MCQ / Short / Long / Chapter explanation), voting eligibility math, flagging queue, comments, moderator promotion logic, strikes, 5-strike auto-ban, CNIC denylist, audit log
- **Observability** — Sentry, Amplitude, Microsoft Clarity (input-masked), OneSignal — all wired before launch, all fail-open on missing env keys
- **Admin tooling** — release-readiness dashboard, AI question generation (BYOK), chapter SEO seed automation, community moderation panels
- **Documentation** — ~84 pages, ~86k words, two Docusaurus sidebars, full SEO/AEO infrastructure
