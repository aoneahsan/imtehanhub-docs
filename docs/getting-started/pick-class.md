# Pick your class — Class 5 through 2nd Year

**Pick your class** is the very first navigation choice on ImtehanHub. It determines which subjects, chapters, and questions you see. The right class is **the class whose textbook you are studying right now**, not the class whose age you happen to be.

> **TL;DR** — Open `/subjects` → pick the class matching your current textbook. If you are between two classes, pick the one whose book you have open in front of you.

## What classes ImtehanHub covers

ImtehanHub serves the **Pakistani national curriculum** from Class 5 through 2nd Year. The full coverage:

| Class | Stage | Typical age | Curriculum |
|---|---|---|---|
| Class 5 | Primary | 10 | Pakistan curriculum |
| Class 6 | Middle | 11 | Pakistan curriculum |
| Class 7 | Middle | 12 | Pakistan curriculum |
| Class 8 | Middle | 13 | Pakistan curriculum |
| Class 9 | Matric | 14 | Pakistan curriculum, board-aligned |
| Class 10 | Matric | 15 | Pakistan curriculum, board-aligned |
| 1st Year (FA / FSc) | Intermediate | 16 | Federal / Punjab / Sindh / KPK / Balochistan / AJK board |
| 2nd Year (FA / FSc) | Intermediate | 17 | Federal / Punjab / Sindh / KPK / Balochistan / AJK board |

Stages mean:

- **Primary (5)** — single track, no streams.
- **Middle (6-8)** — single track, no streams.
- **Matric (9-10)** — board exams begin. You start specialising (Science vs Arts) in Class 9.
- **Intermediate / FA / FSc (1st & 2nd Year)** — formal college years; specialisation by group: FSc-Pre-Medical, FSc-Pre-Engineering, FA, ICS, ICom.

ImtehanHub covers **all groups** within Intermediate. The subject list updates based on the class you pick.

## How to pick

### If you are currently a student

Pick the class **printed on your current textbook's cover**. If the book says "Physics — Class 9", pick Class 9, even if you are 13 or 16.

### If you are revising for a board exam

Pick the class whose **board exam you are sitting**. Class 10 board exam → pick Class 10 even if you are technically still in school. 2nd Year exam → pick 2nd Year.

### If you are between two classes (the common case)

Pick the class whose textbook you are using **for daily homework right now**. If you finished Class 9 last year and are revising before Class 10 starts, you can pick either:

- **Class 9** — to test your retention of last year's material.
- **Class 10** — to preview what is coming next term.

A common pattern: pick Class 9 for revision, take 5-10 tests until you are comfortable, then switch to Class 10 to preview new content.

### If you are a tutor or institute teacher

Pick whichever class your **student** is in. ImtehanHub does not differentiate teacher vs student at the class-pick stage; the difference is in account-level features (institute manager → roster, per-student progress).

## What you see after picking a class

Inside a class you see:

- **Subjects** — the list of subjects taught in that class. Class 9 has English, Urdu, Math, Physics, Chemistry, Biology (or Computer Science), Islamiat, Pakistan Studies. Intermediate adds the group-specific subjects.
- **Chapter counts** — each subject card shows how many chapters and how many MCQs are seeded.
- **Recently updated** — chapters that received new questions in the last week are flagged.

Each class subject opens to a chapter list, and each chapter opens to a "Start Test" page. From there, the [Quick Start](/docs/getting-started/quick-start) flow takes over.

## Switching class later

Picking a class is **not** a permanent choice — it is just a navigation filter. You can switch class any time from the `/subjects` page. The only thing tied to your class is the **add-on Class plan** (PKR 99/mo unlocks one specific extra class for free-tier users); see [Quotas and plans](/docs/concepts/quotas-and-plans) for that.

## Boards and classes

A separate but related choice is your **board** — Punjab, Federal, Sindh, KPK, Balochistan, or AJK. The class structure is the same across boards, but question phrasing and exam style differ. Set your preferred board on the [Profile page](/docs/getting-started/welcome) so questions match your board's wording. See [Pick your board](/docs/getting-started/pick-board) for the breakdown.

## Common questions

### What if my class is not listed?

Currently ImtehanHub covers Class 5 through 2nd Year only. Class 1-4 and university content are out of scope. If you have a serious need, request the content via the [book request flow](/docs/getting-started/welcome) — we keep a public log of requests and prioritise high-demand classes.

### Can I take a higher-class test for practice?

Yes — picking a class is a navigation choice, not a permission gate. Free-tier students can practise any seeded class. The add-on Class plan only matters when you want **unlimited** practice for one specific class beyond the daily quota.

### Do question difficulties scale with class?

Yes. Class 5 questions are mostly recall-and-recognition. Class 9-10 mix recall with application. 1st and 2nd Year tilt toward application and analysis. The difficulty distribution is set per chapter; see [Question types](/docs/concepts/question-types) for how MCQ, Short, and Long differ.

### What about Cambridge / O-Level / A-Level?

Out of scope for now. ImtehanHub focuses on the Pakistani national curriculum (Federal Board) and provincial board variants. Cambridge content has different structure and would need a separate content pipeline.
